Instrucciones de uso

Bienvenido Apicultor! , esta pagina te sera de guia para el manejo de "Bee Project" y su base de datos.
En nuetra pagina de inicio encontraras un pequeño "Login" el cual te permitira el ingreso a tus colmenas, para poder acceder debes escribir correctamente tu usuario y contraseña y dar click al boton "Login".
Una vez dentro podras encontrar los datos de tus colmenas, el cual se iran actualizando en el trascurrir del dia.
El boton de descarga debajo de tu colmena te permitira obtener la base de datos completa de esa colmena, el archivo esta en formato de base de datos pero no te preocupes que te daremos una herramienta que desarrollamos para que puedas obtenerla en excel.
Una vez descargada la base de datos, debes descargar este archivo: sqltoex.exe el cual debes guardar en la misma carpeta donde este el archivo de base de datos, lo ejecutas y listo!
